## Run Locally

Clone the project

```bash
  git clone https://github.com/piyushkhurana8744/mint-pickle-9300.git
```

Go to the project directory

```bash
  cd mint-pickle-9300
```

Install dependencies for frontend as well as backend

```bash
  npm install
```


Start the localhost server

```bash
  npm start
```
